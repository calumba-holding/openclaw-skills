# Exam Preparation Reference

## By Exam Type

### Multiple Choice
- Recognition over recall — know concepts broadly
- Elimination strategies work
- Watch for absolute words ("always", "never")
- First instinct often correct

### Essay
- Thesis-evidence-analysis structure
- Prepare flexible "building blocks"
- Time: outline (5min) → write → review (5min)
- Practice under time pressure

### Oral / Viva
- Practice explaining OUT LOUD
- Prepare for follow-up questions
- Pausing to think is okay

### Practical / Lab
- Procedure requires physical repetition
- Checklists for sequences
- Practice under realistic conditions

---

## Timeline

| Phase | When | Focus |
|-------|------|-------|
| Foundation | 4+ weeks | Understand concepts, fill gaps |
| Practice | 2-4 weeks | Active recall, practice problems |
| Simulation | 1 week | Full practice tests, review mistakes |
| Polish | 2 days | Light review only, sleep |
| Day-of | Exam | Confidence, no new material |

---

## High-Yield Topic Identification

- Past papers reveal 80/20 pattern
- Track topic frequency across 3-5 past exams
- Professor emphasis = likely to appear
- Syllabus objectives = exam blueprint

---

## Day-Of Strategies

**Physical:** Sleep 7-8h, eat protein + carbs, caffeine only if habitual

**Logistics:** Arrive early, bathroom, materials ready

**During exam:**
- Read ALL questions first
- Easiest questions first (momentum)
- Time by points (don't over-invest)
- Skip, mark, return if stuck
- Use full time — review for errors

---

## Managing Anxiety

- Preparation = confidence
- Reframe: "excited" not "anxious"
- Box breathing: 4-4-4-4
- Write worries on paper (expressive writing)
- Worst-case planning removes catastrophizing
