3. Use non-intrusive popups (slide-in, not full-screen)

**Diagnosis**:
- Poor product images
- Missing information
- Unclear pricing
- No reviews

**Solution**:

2. Write detailed descriptions
3. Add size guides/specifications
4. Install review app (Loox)
5. Use popup to highlight free shipping/returns

- Complicated checkout
- No save cart option

1. Show shipping costs early
2. Enable guest checkout

- Unengaged list
