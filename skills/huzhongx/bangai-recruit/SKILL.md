# Bang.AI 招聘平台

7x24小时 AI 自动招聘平台，支持自动搜寻候选人、确认意向、沟通需求、生成 AI 视频简历。

## 平台入口

- URL: https://bangai.hina.com/agent?tab=my-interviews
- 需要登录（微信扫码或手机验证码）

## 主要功能

### 1. 查看招聘Agent列表

访问招聘Agent页面，查看所有在招岗位及其状态。

每个Agent显示：
- 岗位名称、薪资范围
- 学历/经验要求
- 已投递/沟通中/沟通成功 数量

### 2. 查看候选人列表

点击顶部"候选人"标签。

候选人信息包括：
- 姓名、应聘岗位、来源
- 学校、专业、学历
- 工作经验、年龄
- 投递时间、沟通状态
- AI 视频简历（如有）

### 3. 筛选候选人

可按以下条件筛选：
- 岗位、面试状态、院校、学历、工作年限、年龄、性别、来源

### 4. 审核候选人

对于每个候选人卡片：
- 点击"通过"按钮
- 点击"拒绝"按钮
- 点击候选人卡片查看详情和视频简历

### 5. 创建招聘Agent

在招聘Agent页面创建新的AI招聘代理。

操作步骤：
1. 填写需求描述框（系统会自动解析）
2. 填写表单字段：岗位名称（必填）、工作年限、学历、技能等
3. 点击"开始帮我招人"按钮
4. Agent自动工作：搜寻候选人 -> 确认意向 -> 沟通需求 -> 生成视频简历

## 注意事项

- 需要用户已登录 Bang.AI
- 审核候选人前建议先查看视频简历
- Agent 沟通是自动进行的，无需人工干预
- 通过的候选人可以进行后续线下面试
