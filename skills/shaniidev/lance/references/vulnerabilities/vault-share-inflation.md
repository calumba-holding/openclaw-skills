# Vault Share Inflation

## Hunt Targets

- first-deposit edge cases
- donation/inflation paths in share conversion
- virtual share/asset offset absence

## Exploit Checks

- simulate first depositor advantage or donation skew
- show victim receives unfairly low shares or attacker extracts excess assets

## Reject Conditions

- protocol uses proven anti-inflation offsets
- exploit does not create extractable value

## Evidence Required

- share conversion math
- attacker and victim balance delta
