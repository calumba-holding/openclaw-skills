# Changelog

All notable changes to `lance` are documented in this file.
Versioning follows Semantic Versioning.

## [0.0.1] - 2026-02-25

### Added
- Initial Lance skill release.
- Core 7-gate workflow (scope, intake, detection, exploitability, economics, FP elimination, triage/reporting).
- Web3 references for exploit validation, economic feasibility, severity, and triage simulation.
- Chain guidance for EVM, Sui Move, bridge, and L2 contexts.
- Platform guidance for Immunefi, HackenProof, HackerOne Web3, and Bugcrowd Web3.
- Vulnerability playbooks covering high-value Web3 classes.
- Script suite for scope parsing, target normalization, scoring, adapter conversion, triage simulation, and report generation.
- Finding schema and platform report templates.
