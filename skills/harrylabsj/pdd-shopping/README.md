# PDD Shopping Skill v2.0

拼多多购物助手 Skill，支持浏览器自动化完成搜索、百亿补贴、拼团、加购物车等操作，支付环节留给用户控制。

## 功能特性

### v2.0 - 新增浏览器自动化

- 🔍 **商品搜索**：关键词搜索，支持筛选
- 💰 **百亿补贴**：浏览平台补贴商品
- 👥 **拼团**：发起或加入拼团
- 🏪 **卖家验证**：检查店铺评分、徽章、评价
- 🛒 **购物车操作**：登录态下添加商品
- 🎫 **优惠券**：自动检查平台/卖家优惠券
- 📋 **订单预览**：生成完整订单信息（不含支付）

### v1.0 - 决策支持

- ✅ 拼团策略指南
- ✅ 卖家验证技巧
- ✅ 质量评估建议
- ✅ 百亿补贴导航
- ✅ 分类购买建议

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 查看百亿补贴 | ✅ | - |
| 发起/加入拼团 | ✅ | - |
| **支付/提交订单** | ❌ | ✅ |

## 工作流程

```
发现 → 卖家验证 → 选择 → 加购/拼团 → 用券 → 预览 → [用户支付]
```

## 版本历史

- v1.1.1 - 基础购物指南

## 安装

```bash
clawhub install pdd-shopping
```

## 使用示例

```
用户：帮我买拼多多的耳机
Agent：搜索 → 验证卖家 → 对比拼团价 → 发起拼团 → 生成订单预览
用户：👤 完成支付
```
