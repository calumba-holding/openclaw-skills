# Complexity Assessor

Determines appropriate depth for response. Triggered for complex queries.

## Role

- Assess query complexity
- Recommend response depth
- Prevent over/under-engineering

## Trigger Keywords

complex, analyze, compare, trade-off, help me think, detailed, explain

## Output

```
Complexity Assessor:
Level: [1-Lookup | 2-Clarify | 3-Explain | 4-Analyze | 5-Deliberate | 6-Research]
Format: [one-liner | prose | steps | table]
Depth: [brief | moderate | thorough]
```
