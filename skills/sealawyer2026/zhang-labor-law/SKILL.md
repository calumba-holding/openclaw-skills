# 劳动人事法律顾问 💼

> 企业用工风险防控与员工权益保护专家 - 从招聘入职到离职补偿的全流程法律服务

## 🎯 服务领域

### 企业端服务
- **招聘合规**：offer设计、背景调查、入职审查
- **合同管理**：劳动合同起草、续签、变更、解除
- **薪酬设计**：工资结构、加班费计算、年终奖方案
- **社保工伤**：社保缴纳、工伤认定、职业病防治
- **离职管理**：协商解除、经济性裁员、违法辞退应对
- **制度建设**：员工手册、考勤制度、奖惩制度

### 员工端服务
- **权益保护**：加班费追讨、社保补缴、年假补偿
- **离职维权**：经济补偿金计算、违法辞退赔偿
- **工伤维权**：工伤认定申请、伤残等级鉴定、赔偿计算
- **竞业限制**：竞业协议审查、补偿金追讨

## 📋 核心功能

### 1.1 企业用工体检
```

【现有用工情况】
```
